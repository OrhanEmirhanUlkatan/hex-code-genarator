
# Project Title

Hex Code Generator

## Run Locally

Clone the project

```bash
  git clone https://github.com/OrhanEmirhanUlkatan/hex-code-generator.git
```

Go to the project directory

```bash
  cd hex-code-generator
```

Install dependencies

```bash
  npm install || yarn
```

Start the server

```bash
  npm start || yarn start
```
